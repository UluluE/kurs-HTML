id используют что бы уникально выделить элемент
class использую что бы придать определенную стилистику

div блочный эллемент
span строчный элемент

что бы быстро создать div элемент с помощью emmt можно использовать . и #

  • "div.+tab" c class
  • "span.+tab" c class
  • "div#(название id)" с id
  • "ul.list" c class
  • "ul#(название id)" с id
  • "." div
